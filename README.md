# Plant Watering Tracker

A simple Django web application to help users track their plants and watering schedules. This project demonstrates basic CRUD functionality in Django, making it a great practice project for beginners.

---

## Features

- Add new plants with details like name, species, last watered date, and notes.
- View a list of all plants with their details.
- Edit or delete plant information as needed.
- Simple and clean interface for easy use.
- Optional enhancements: user authentication, reminders, and search functionality.

---

## Technologies Used

- **Backend**: Django (Python)
- **Frontend**: HTML, CSS (Bootstrap for styling)
- **Database**: SQLite (default Django database)
- **Deployment**: (Optional) Gunicorn and Heroku/AWS

---

## Getting Started

### Prerequisites

- Python 3.8+
- pip (Python package manager)
- Conda environment (optional, but recommended)

### Installation

1. Clone the repository:
   
   ```bash
   git clone https://github.com/your-username/plant-watering-tracker.git
   cd plant-watering-tracker
2. Produce and activate a conda environment:
   
   ```bash
   conda create --name <environment_name> python=3.12 
   conda activate <environment_name>
3. Install the dependencies:

   ```bash
   pip install -r requirements.txt
